Injecting the <meta http-equiv="origin-trial" …> tag with JavaScript

↰ All demos

Code for this demo

The top-level page includes JavaScript which injects the <meta http-equiv="origin-trial"…/> tag: 

  const otMeta = document.createElement('meta');
  otMeta.httpEquiv = 'origin-trial';
  otMeta.content = '{{token_1p}}';
  document.head.append(otMeta);

This uses a first-party token for ps-origin-trial.glitch.me.
