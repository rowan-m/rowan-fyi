Privacy Sandbox origin trial demo

Injecting the <meta http-equiv="origin-trial" …> tag with third-party JavaScript

↰ All demos

Code for this demo

The top-level page includes a cross-origin JavaScript file which injects the <meta http-equiv="origin-trial"…/> tag: 

  const otMeta = document.createElement('meta');
  otMeta.httpEquiv = 'origin-trial';
  otMeta.content = '{{token_1p}}';
  document.head.append(otMeta);

This uses a third-party token for x-origin-src.glitch.me.
